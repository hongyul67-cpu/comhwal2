🛠️ 시스템 관리 & 보안

⚠️ 자주 나오는 함정

  • 드라이브 조각 모음 = 접근 속도↑ (용량 늘어남 ✗) / 디스크 정리 = 불필요 파일 삭제로 공간 확보
  • 조각 모음 대상 아님: CD-ROM · 네트워크 드라이브 (SSD는 TRIM 대상)
  • 기본 프린터는 한 대만 지정 · 네트워크 프린터도 가능 · 삭제 가능
  • 스풀(SPOOL): 인쇄 중 다른 작업 가능하지만 전체 인쇄 속도는 느려짐
  • 레지스트리=환경설정 DB(regedit) · 잘못 수정 시 데이터 손실도 가능
  • 공개키(RSA)=비대칭·키 관리 쉬움·속도 느림 / 비밀키=대칭·빠름·키 관리 어려움
  • 방화벽=외부 침입 차단 (내부 해킹은 못 막음)

🗂️ 데이터 관리 & 분석

⚠️ 자주 나오는 함정

  • 정렬: 최대 64개 기준 · 셀 값·셀 색·글꼴 색·아이콘 · 빈 셀은 항상 맨 뒤 · 숨겨진 행/열 제외
  • 고급 필터: 같은 행=AND(그리고) · 다른 행=OR(또는)
  • 자동 필터: 여러 필드는 AND · 결과를 다른 곳 추출·OR 연결 불가(고급 필터의 기능)
  • 부분합: 반드시 그룹 기준 필드 정렬 먼저 · 중첩 시 '새로운 값으로 대치' 해제 · 중앙값·최빈값 함수 없음
  • 피벗 테이블: 원본 바뀌면 [새로고침] 해야 반영 · 새 시트에도 생성 가능
  • 목표값 찾기: 값을 바꿀 셀은 반드시 1개 / 시나리오: 변경 셀 최대 32개, 요약은 자동 갱신 안 됨
  • 데이터 표: 결과는 배열 → 부분 수정·삭제 불가
